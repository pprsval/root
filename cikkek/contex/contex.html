Klasszikus helyi menü visszaállítás Windows 11 rendszeren (és fordítva)

Forrásként felhasználtam: Restore old Right-click Context menu in Windows 11

A Windows 11 rendszer sok mindenben megváltozott elődjéhez képest, többek között a jobb egér kattintással előhívható, úgynevezett helyi menü (pic. 1.). Ez egy olyan változtatás, ami egyesek számára – magamat is beleértve - kényelmetlenné teszi a gördülékeny munkavégzést.

PIC. 1.
Új helyi menü
Klasszikus helyi menü

Állítsuk hát vissza a régi helyi menüt:

1. Hozzuk be a terminált: jobb klikk a START gombra, majd klikk a TERMINÁL -ra.

2. Másoljuk az alábbi parancsot, majd illesszük be a terminálba és nyomjunk ENTERT.

reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve

Megjegyzés:
Ha a klasszikus Helyi menüt szeretnék Modernre változtatni, akkor az alábbi parancsot illesszük be a Terminálba:

reg.exe delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f

3. Indítsuk újra a FILE EXPLORER -t vagyis a WINDOWS INTÉZŐT:
Nyissuk meg a FELADATKEZELŐT (CTRL + SHIFT + ESC),
Jobb klikk a WINDOWS INTÉZŐRE,
Újraindítás.

És készen vagyunk.
